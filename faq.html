FAQ

Quick answers before someone leaves

This page removes basic friction: where to buy, whether there is a demo, supported platform, local multiplayer, and input support.

Where do I buy it?

The current full version is sold on itch.io. The website should point there for payment and file delivery.

Is there a free version?

Yes. There is a free demo on the itch.io page, with the core gameplay available now.

What platform is supported?

The current public release is for Windows.

How many players?

The public listing supports one to two players locally.

Does it have co-op and PvP?

Yes. Solo, local co-op, and local PvP are all part of the current release.

What inputs are supported?

Keyboard, mouse, and gamepad support are listed publicly, along with configurable controls.
