Game systems

Move fast. Read the maze. Stay alive.

The game starts with a simple loop, then layers pressure on top of it. Movement is the core, but the route keeps changing because the run keeps changing.

Store page See videos
Core mechanics

Dash and phase are the identity.

⚡

Dash

Boost forward, escape danger, and pressure other players in PvP. Dash upgrades can improve cooldown, duration, and extra boons.

🧱

Phase

Go through walls and stretch your route options. Phase and dash can stack together for extra travel distance.

🎯

Optimization

The best runs come from better routing, cleaner execution, and faster adaptation as the floor changes.

Pressure

Hazards, powerups, and bosses keep the pace up.

Powerups

Shielding, fire trails, and weapons change how a run feels and how a fight plays out.

Hazards

Goo slows you down, hot floors burn, lava punishes harder, and moving walls block clean routes.

Bosses

Random boss encounters break the flow in a good way and force more adaptation.

Fog + minimap

You still need to read the board. Navigation matters, not just raw speed.

Customization + tools

Build weird maps. Change the look. Tweak the sound.

Map maker

Create and test your own mazes

Build a map, install it into Crum Run, then run it from the start of the game. The full version adds save/load/share support.

Emoji sprites

Swap the look of the game

The game uses emoji-based visuals, so unlocking more assignments changes the whole feel of a run.

Hats

Cosmetic unlocks

Find hats, then adjust size, rotation, and placement to make the characters feel more personal.

DSP sound

Custom sound scheme system

Tune the sound side of the game and save it in the full version alongside the rest of your custom setup.
