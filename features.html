Feature breakdown

What the game actually includes

This page keeps the strongest public details from the store page, but reorganizes them into something easier to scan.

Core identity

Movement, routing, and replayability

Dash

Dash boosts move speed and throws the player forward. It supports cooldown and duration upgrades and can become an offensive threat in PvP.

Phase

Phase pushes the character through walls. Combined with Dash, it opens longer lines, safer escapes, and more aggressive route choices.

Endless pressure

Runs scale upward with increasing difficulty, forcing cleaner movement, quicker reads, and stronger adaptation to hazards and layouts.

Random variety

Boss encounters, floor hazards, powerups, and procedural maps help keep sessions from settling into one static pattern.

Systems

Powerups, obstacles, and support

Powerups and weapons

Run modifiers include effects and weapons such as fire trail, mega shield, rapid blaster, snake cannon, and shotgun.

Hazards

Goo, hot coals, lava, moving walls, fog of war, and other floor elements force route changes and create pressure.

Controls and display

Full key remapping, USB gamepad support, keyboard and mouse input, and adjustable display settings are already part of the release.

Customization

More than a basic score attack game

Map maker

Create custom maze maps, test them, then save, load, and share them in the full version.

Emoji visuals

The game uses emoji-based sprites so players can change how the game looks and eventually unlock custom emoji assignment.

Hats and player identity

Unlock hats and place them with custom size, rotation, and positioning. Player naming also feeds into score and PvP identity.

Custom sound tools

The current public listing calls out a custom DSP sound scheme system and saving/sharing support in the full version.

Version comparison

Demo and full version

  • Core gameplay included
  • Run ends at floor 11
  • Some customization features stay locked
  • PvP reduced from the full version setup

Full version

  • Unlimited floor progress
  • Unlimited level progress
  • Save, load, and share maps
  • Save, load, and share sound settings
  • Save, load, and share custom emoji sets once unlocked
